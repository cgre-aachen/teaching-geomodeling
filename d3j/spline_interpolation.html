Cubic Spline Interpolation - Theory

Cubic spline interpolation is a method of curve interpolation using cubic polynomials. It is commonly used to construct smooth curves that pass through a given set of points.

A cubic spline is a piecewise polynomial function of degree 3 that is continuous and has continuous first and second derivatives. The function is defined by a set of control points, and between each pair of control points, the function is a cubic polynomial.

The goal of spline interpolation is to find a spline function that smoothly interpolates between the given data points.

The general form of a cubic polynomial is:

\( S(x) = a_ix^3 + b_ix^2 + c_ix + d_i \)

where \( S(x) \) is the spline function, and \( a_i, b_i, c_i, \) and \( d_i \) are the spline coefficients for the ith interval.
